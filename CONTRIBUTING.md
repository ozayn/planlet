# Contributing to Planlet

## Design

All UI, UX, and interaction work should follow:

**[docs/DESIGN_PHILOSOPHY.md](./docs/DESIGN_PHILOSOPHY.md)**

That document is the default reference. Before finishing a UI change, run the review checklist there (remove, dedupe, content-first, mobile, consistency, cognitive load).

Also see:

- [docs/BRANDING.md](./docs/BRANDING.md) — logo and color tokens
- [docs/PRODUCT_NOTES.md](./docs/PRODUCT_NOTES.md) — product scope notes
- `.cursor/rules/planlet-ui.mdc` — always-on agent UI preferences

## Development

See [README.md](./README.md) for setup, scripts, and environment variables.
